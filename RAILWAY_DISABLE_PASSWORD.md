# Отключение проверки пароля на Railway

## Что сделано:

Добавлена возможность отключить проверку пароля через переменную окружения `DISABLE_PASSWORD_CHECK`.

---

## Настройка на Railway:

### Backend Service → Variables:

Добавьте переменную:

```
DISABLE_PASSWORD_CHECK=true
```

---

## Как это работает:

- Если `DISABLE_PASSWORD_CHECK=true` - пароль не проверяется, токен создается для любого username
- Если `DISABLE_PASSWORD_CHECK=false` или не установлено - работает обычная проверка пароля

---

## ВАЖНО:

⚠️ **Это небезопасно для production!** Используйте только для отладки или если действительно нужно отключить проверку пароля.

---

## После добавления переменной:

1. Перезапустите Backend Service на Railway
2. Попробуйте войти с любым username (пароль не важен)
3. Должен работать вход!
